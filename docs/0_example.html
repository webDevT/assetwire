Formatting Dates

Dates in your import file must be a formatted date value in Excel. Dates must have the format YYYY-MM-DD HH:mm.

This applies to Date Performed, Next Event Date and form criteria that are setup as a date type.

Are signature criteria types supported?

No. The event importer does not support the signature data type. Signatures must be signed by editing the imported event and signing on screen.

What template should I use?

If you record Recommendations or Deficiencies during your events then use the template with Observations. Both templates will include all event form (checklist) criteria. You are not required to fill out the event form for the import.

Is there a maximum number of records I can import?

Yes. We recommend a maximum of 5000 rows per import. For larger imports you can split the import into multiple files.
