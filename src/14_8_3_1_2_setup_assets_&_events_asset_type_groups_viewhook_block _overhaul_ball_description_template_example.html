Asset Description Template Formatting

The Asset Description template is used throughout Assetwire Web & Mobile to provide a description of each asset. You can customize how this looks for each Asset Type. Sentences are created with a combination of preconfigured and custom notations.

There is no limit to the number of notations you can use and each notation can be separated by any character or word.

Notation

{Identifier}

{RFID}

{RefNumber}

{OrderNumber}

{PONumber}

{PackingSlips}

{5MillionNumber}

{CustomAttributeName}

Displays

The Identifier of the asset

The RFID Number on the asset

The Reference Number on the asset

The Order Number on the asset

The Purchase Order on the asset

The Packing Slips on the asset

The 5 Million Number on the asset

The value of the attribute specified

Examples
will display:

SN421 (E00f01000089EE5B8), 1/4" X 42ft, BD2, 24,000lbs

will display:

Manufactured by ACME on 12/12/11 with a 2 year warranty
