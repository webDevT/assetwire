Formatting Dates

Dates in your import file must be a formatted date value in Excel. Dates must have the format YYYY-MM-DD HH:mm.
This applies to Identified Date and Asset Attributes that are setup as a date type.

How long will the import take?

This depends on how many records you are importing. The import process will take anywhere from a few minutes up to 20 minutes.

Is there a maximum number of records I can import?

Yes. We recommend a maximum of 5000 rows per import. For larger imports you can split the import into multiple files.

Can I delete records through the import?

No. The import process will only add and update your data. You can never delete anything through the import process.
