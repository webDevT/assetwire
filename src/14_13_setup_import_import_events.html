Import Events

1. Select Event Type

First, select the Asset Type you would like to import. You will do a separate import for each Asset Type.

2. Download Template

This template is built specifically for the Event Type selected above. Download the template and follow the format to build your import file.

Download Excel (XLSX) Import Template with Observations
Download Excel (XLSX) Import Template without Observations
3. Upload

Upload your completed Excel import file below. Your file will be processed for errors before the import begins. The correct Asset Type must be selected above for the import to run successfully.

*Warning*- Do not change the formatting of the Excel template! (ex:moving columns or renaming the header) This will cause the import to fail.*

Drop files here or click to upload.

  •  

Formatting Dates

Dates in your import file must be a formatted date value in Excel. Dates must have the format YYYY-MM-DD HH:mm.
This applies to Identified Date and Asset Attributes that are setup as a date type.

Are signature criteria types supported?

No. The event importer does not support the signature data type. Signatures must be signed by editing the imported event and signing on screen.

What template should I use?

If you record Recommendations or Deficiencies during your events then use the template with Observations. Both templates will include all event form (checklist) criteria. You are not required to fill out the event form for the import.

Is there a maximum number of records I can import?

Yes. We recommend a maximum of 5000 rows per import. For larger imports you can split the import into multiple files.
